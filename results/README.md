# Results Directory

This directory contains the output artifacts generated by the deepfake detection evaluation and benchmarking pipeline.

## Structure

```
results/
├── evaluation/           # Comprehensive evaluation outputs
│   ├── model_comparison.csv       # Standardized timm-style benchmark results
│   ├── detailed_results.json      # Comprehensive metric breakdowns
│   └── explainability/            # Grad-CAM attention heatmaps
│       ├── explainability_vit.png
│       ├── explainability_deit.png
│       └── explainability_swin.png
├── benchmarks/           # Standalone benchmarking logs
└── training/             # Training logs and validation metrics
```

## Generated Artifacts

### 1. `model_comparison.csv`
A detailed, standard CSV summarizing the performance of the Vision Transformers (ViT, DeiT, Swin) alongside the Stacked Ensemble.
Contains metrics such as accuracy, precision, recall, f1-score, AUC, and inference timings.

### 2. `detailed_results.json`
A deep-dive statistical analysis and component-wise breakdown, used primarily by the `notebooks/analysis.ipynb` for plotting graphs, rankings, and deep insight analysis.

### 3. `explainability/` (Grad-CAM Visualizations)
Contains visual overlays demonstrating where each transformer model focused its attention when determining if a video frame was real or deepfake. The visual gradients highlight facial features critical to the decision-making process.

> **Note:** Unlike the original PyTorch Image Models (`timm`) repository which is evaluated against ImageNet datasets, our framework evaluates exclusively and explicitly against the testing splits of FaceForensics++ and CelebDF. All ImageNet logic and results have been completely phased out.
